# Simple Calculator

### Description
A web based calculator application.

# How To Play
* Rock beats Scissors
* Scissors beats Paper
* Paper Beats Rock
* First to five wins

### Functionality
* Buttons for choice selection
* Display round based results
* Display winner result

### Code Base
* JavaScript
* HTML
* CSS
